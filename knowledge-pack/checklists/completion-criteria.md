# Checklist: Completion Criteria

## Outcome

- [ ] 目标达成。
- [ ] scope 内事项完成。
- [ ] 不在 scope 的事项未误改。

## Verification

- [ ] 已运行与风险匹配的验证。
- [ ] 失败检查已解释。
- [ ] 未运行检查有原因。
- [ ] regression 风险已处理。

## Evidence

- [ ] 关键结论有来源、trace、test 或 artifact。
- [ ] 重要文件和变更已列出。
- [ ] residual risks 已记录。

## Handoff

- [ ] 当前状态清楚。
- [ ] 下一步清楚。
- [ ] known failures 清楚。
- [ ] 其他 agent 可继续。

## Governance

- [ ] 没有未授权动作。
- [ ] secrets 未泄露。
- [ ] audit/trace 足够。
- [ ] 高风险动作有审批或明确理由。
