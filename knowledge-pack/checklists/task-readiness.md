# Checklist: Task Readiness

## Objective

- [ ] 目标明确。
- [ ] scope 明确。
- [ ] success criteria 明确。
- [ ] done/not done 可判断。

## Evidence

- [ ] 已识别需要的来源或代码上下文。
- [ ] 重要结论有来源或验证路径。
- [ ] 不确定点已标记。

## Harness Layers

- [ ] Execution 环境已知。
- [ ] Tools 可用且权限明确。
- [ ] Context/memory 初始状态明确。
- [ ] Lifecycle plan 已写。
- [ ] Observability/trace 方案已知。
- [ ] Verification plan 已写。
- [ ] Governance constraints 已知。

## Risk

- [ ] destructive/external/expensive actions 已识别。
- [ ] credential/secrets 风险已处理。
- [ ] rollback/recovery 已定义。
- [ ] human escalation 条件已定义。

## Go/No-Go

- [ ] 可以开始执行。
- [ ] 需要先补证据。
- [ ] 需要人类确认。
